Data Analyst

As a data analyst for OpenSOS, you will need to work efficiently with our team to collect and meaningfully present data in multiple areas of interest, including emerging issues with our technologies, how users interact with our programs, and how our work can be made more effective.

Key Responsibilities
  1. Collecting data from reliable sources
  2. Cleaning data and interoperating it in a meaningful way
  3. Presenting data in a clear way that is easy to understand
Key Attributes
  • Essential
    • 1 Year of experience as a data analyst
    • A Degree in Computer Science or a related field
  • Preferred
    • Experience using Excel, Python, and SQL\
    • Good communication skills
    • A positive attitude towards learning in the workspace and striving to improve

Salery: $90000 - $100000 per annum

Reports to: Analytics Leader

Reference Number: H3110

Programmer

As a programmer for OpenSOS, you will be able to produce code necessary for a variety of programs, working closely with our software development team to produce the best solutions possible. You will be expected to write and review your own code, making it run optimally, and for its purpose and functionality to be clear to your team.

Key Responsibilities
  1. Write clear and efficient code
  2. Test code to ensure reliability across different environments
  3. Maintain and update softwares
Key Attributes
  • Essential
    • 1 Year of experience as a programmer
    • A Degree in Computer Science or a related field
  • Preferred
    • Experience using programming languages such as c#, c++ and python
    • A background of previously worked-on programs
    • Good communication skills
    • A positive attitude towards learning in the workspace and striving to improve

Salery: $100000 - $120000 per annum

Reports to: Director of Software Engineering

Reference Number: T4B13

Front-end Web Developer

As a front-end web developer for OpenSOS, you will work to produce effective web solutions for various applications. You will develop web-pages alongside back-end developers and designers, and must be able to create clean interfaces and transfer user data securely. You will follow designs to produce well-structured UI for various needs.

Key Responsibilities
  1. Create clean UI for web pages
  2. Ensure websites work, and are usable in a range of different environments and with a variety of users
  3. Collaborate with designers and back-end developers effectively
Key Attributes
  • Essential
    • 1 Year of experience as a front-end developer
    • A Degree in Computer Science or a related field
    • Experience using HTML, CSS and JavaScript
  • Preferred
    • A background of previously worked-on web pages
    • Good communication skills
    • A positive attitude towards learning in the workspace and striving to improve

Salery: $90000 - $110000 per annum

Reports to: Web development manager

Reference Number: P1224
